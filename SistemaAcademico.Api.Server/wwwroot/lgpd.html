Privacidade e segurança

Privacidade e Proteção de Dados

A privacidade e a segurança das informações dos usuários são prioridades no BioTecher. Todos os dados utilizados na plataforma são tratados com responsabilidade, confidencialidade e em conformidade com os princípios da Lei Geral de Proteção de Dados (LGPD - Lei nº 13.709/2018).

Coleta de informações

O BioTecher poderá coletar informações necessárias para identificação, autenticação e acompanhamento dos serviços laboratoriais, incluindo:

  • nome completo;
  • e-mail;
  • informações de acesso;
  • dados relacionados às solicitações médicas;
  • exames laboratoriais;
  • status de acompanhamento e resultados.

As informações coletadas são utilizadas exclusivamente para fins operacionais, assistenciais e de gerenciamento interno da plataforma.

Uso das informações

Os dados cadastrados são utilizados para:

  • autenticação segura de usuários;
  • gerenciamento de solicitações laboratoriais;
  • acompanhamento do processamento de exames;
  • disponibilização de resultados;
  • comunicação entre paciente, laboratório e equipe responsável.

O BioTecher não realiza comercialização ou compartilhamento indevido de informações pessoais.

Segurança das informações

Para garantir maior proteção aos dados armazenados, o sistema utiliza mecanismos de segurança, incluindo:

  • autenticação por login e senha;
  • validação de acesso via token JWT;
  • armazenamento seguro de credenciais;
  • controle de acesso às áreas internas da plataforma;
  • monitoramento básico de autenticação e sessão.

Direitos do usuário

Os usuários podem solicitar atualização ou correção de informações cadastrais sempre que necessário, garantindo maior transparência e controle sobre seus dados pessoais.

Compromisso com a privacidade

O BioTecher mantém o compromisso de utilizar as informações de forma ética, segura e responsável, buscando continuamente boas práticas de proteção de dados e segurança da informação para oferecer uma experiência confiável aos usuários da plataforma.

Informação laboratorial clara, acompanhamento seguro e comunicação simples entre paciente e laboratório.
